    Méthodes d'instance

    save(<$db>)

    Enregistre en base de données un nouvel enregistrement

    update(<$db>)

    Mise a jour des données d'un enregistrement

    Méthodes de classe

    Media_modelFromRow(<$row>)

    Crée une nouvelle instance à partir d'un enregistrement de base de données

    getAllMedias(<$db>, <$orderBy = null>, <$asc = null>, <$limit = null>, <$offset = null>)

    récupère tous les enregistrements
    • $db : Base de données
    • $orderBy : clé de tri
    • $asc : 'asc' ou 'desc'. Par défaut : 'asc'
    • $limit : nombre d'enregistrements limite
    • $offset : nombre d'enregistrements sauter avant de récupérer le premier enregistrement

    getCountMedias(<$db>)

    Compte le nombre d'enregistrements
    • $db : Base de données

    getMedia(<$db>, <$id>)

    Récupère un objet par la clé d'unicité

    delete(<$db>, <$id>)

    Supprime un objet par la clé d'unicité

    getAllByCriteria(<$db>, <$criteriaArray>, <$orderBy = null>, <$asc = null>, <$limit = null>, <$offset = null>)

    Récupère les objets selon les critères de sélection
    • $db : Base de données
    • $criteriaArray : Liste de critères de sélection
    • $orderBy : clé de tri
    • $asc : 'asc' ou 'desc'. Par défaut : 'asc'
    • $limit : nombre d'enregistrements limite
    • $offset : nombre d'enregistrements sauter avant de récupérer le premier enregistrement

    countByCriteria(<$db>, <$criteriaArray>)

    Compte le nombre d'enregistrements selon les critères de sélection
    • $db : Base de données
    • $criteriaArray : Liste de critères de sélection
