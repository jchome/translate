SuccessOnContent_model

Variables d'instance

id : type int

Identifiant système

user_id : type int

Lien vers l'utilisateur

content_id : type int

Lien vers le contenu

quotation : type int

Entre 0 et 100, niveau de succès du contenu. Si vide, l'utilisateur n'a pas encore noté mais il l'a commencé.

dateRealization : type date(dd/mm/yyyy)

Date de réalisation par l'utilisateur

xp : type int

Nombre de XP gagnés

gems : type int

Nombre de gems gagnés

evaluation_answer_id : type int

Lien vers la réponse à l'évaluation

root_content_id : type int

Lien vers le contenu racine

Variables de classe

Méthodes d'instance

save(<$db>)

Enregistre en base de données un nouvel enregistrement

update(<$db>)

Mise a jour des données d'un enregistrement

Méthodes de classe

SuccessOnContent_modelFromRow(<$row>)

Crée une nouvelle instance à partir d'un enregistrement de base de données

getAllSuccessOnContents(<$db>, <$orderBy = null>, <$asc = null>, <$limit = null>, <$offset = null>)

récupère tous les enregistrements
  • $db : Base de données
  • $orderBy : clé de tri
  • $asc : 'asc' ou 'desc'. Par défaut : 'asc'
  • $limit : nombre d'enregistrements limite
  • $offset : nombre d'enregistrements sauter avant de récupérer le premier enregistrement

getCountSuccessOnContents(<$db>)

Compte le nombre d'enregistrements
  • $db : Base de données

getSuccessOnContent(<$db>, <$id>)

Récupère un objet par la clé d'unicité

delete(<$db>, <$id>)

Supprime un objet par la clé d'unicité

getAllByCriteria(<$db>, <$criteriaArray>, <$orderBy = null>, <$asc = null>, <$limit = null>, <$offset = null>)

Récupère les objets selon les critères de sélection
  • $db : Base de données
  • $criteriaArray : Liste de critères de sélection
  • $orderBy : clé de tri
  • $asc : 'asc' ou 'desc'. Par défaut : 'asc'
  • $limit : nombre d'enregistrements limite
  • $offset : nombre d'enregistrements sauter avant de récupérer le premier enregistrement

countByCriteria(<$db>, <$criteriaArray>)

Compte le nombre d'enregistrements selon les critères de sélection
  • $db : Base de données
  • $criteriaArray : Liste de critères de sélection
